# TrabalhoSistemasOperacionasiCS# TrabalhoSistemasOperacionasiCS

Desenvolvido para a aula de Sistemas Operacionais na faculdade. 

O projeto tem como objetivo fazer um programa com poucas funções em qualquer linguagem. O programa tem as opções de verificar a versão do arquivo, help, abrir o navegador chrome, criar um diretório e criar um arquivo.

A linguagem escolhida foi o C# no modo de Console.
